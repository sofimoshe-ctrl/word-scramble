# სიტყვების გაშლის რბოლა 🔤

Terminal-ში სიტყვების გაშლის თამაში ქართულ-ინგლისური ლექსიკით.

## გაშვება

```bash
python3 game.py
```

## თამაშის წესები

- აირჩიე თემა: ცხოველები, ქალაქები ან საჭმელი
- თითოეული სიტყვა გაშლილია — გამოიცანი ორიგინალი
- თითო სიტყვაზე 15 წამი გაქვს
- სწორი პასუხი ქულას გაძლევს

## ფუნქციები

- 3 თემა, 12 სიტყვა თითოში
- ფერადი Terminal-ის გრაფიკა
- ლიდერბორდი (ინახება `scores.json`-ში)
- ქრონომეტრი

## მოთხოვნები

- Python 3.6+
